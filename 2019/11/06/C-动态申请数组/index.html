C++动态申请数组

1.使用new运算符来申请数组更加简便，同时也会使指针更像数组。
2.每一次new操作都会向内存申请一块连续的地址空间，也就是说二维数组相邻的列在内存中地址不一定是相邻的。
3.C++中申请内存不必区分基本类型还是自定义的类，都是统一的定义方法。
