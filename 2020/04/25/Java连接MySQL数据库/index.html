1 下载Java驱动

官网下载地址：https://dev.mysql.com/downloads/connector/j/
选择 PlatformIndependent 平台无关版本下的ZIP格式


点击直接下载

只需要 mysql-connector-java-****.jar 文件，把该文件放到固定的目录下
